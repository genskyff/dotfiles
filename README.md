# Dotfiles

## Usage

1. Clone this repository.

    ```shell
    git clone https://github.com/genskyff/dotfiles.git && cd dotfiles
    ```

2. Run the `setup` script to install necessary packages and configure settings.

    - Windows

        ```shell
        .\setup.ps1
        ```

    - macOS / Arch / Debian

        ```shell
        ./setup.sh
        ```

3.   Reload config.

     -   PowerShell

         ```shell
         . $profile
         ```

     -   Bash

         ```shell
         . ~/.bashrc
         ```

     -   Zsh

         ```shell
         . ~/.zshrc
         ```
